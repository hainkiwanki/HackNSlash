# Project_HackNSlash
 Personal game project heavily inspired by games such as Diablo3, Path of Exile, Dungeon Siege I and II
 
# Logs

## August 2019

- __Start of the project__
    - Setting up scene: Simple terrain mesh with a nav mesh
    - Camera behavior: After a lot of testing and switching between setups, I have decided that the isometric perspective is the winner for now.
    - Player movement: Player can move by left-clicking or by holding left-click. Player is an AI agent on a nav mesh.
    - Input: Left and right mouse click, left mouse hold. And some keys
- __Inventory System__
    - Grid based *18/08/2019*
    - Can add items *18/08/2019*
